# ggsegHO: Harvard-Oxford Atlas for the 'ggseg' Ecosystem

Harvard-Oxford cortical atlas for the 'ggseg' ecosystem. Provides a
unified 'ggseg_atlas' object with 2D polygon geometry for use with
'ggseg'.
